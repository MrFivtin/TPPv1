ЭЛЕКТРИЧЕСКИЙ ТОК: ФОРМУЛЫ 11 КЛАСС

Основные характеристики тока

I = q / t (сила тока)
I = n·e·v·S (для металлов)
j = I / S (плотность тока)
e = 1.6·10⁻¹⁹ Кл (заряд электрона)

где I - сила тока [А], q - заряд [Кл], t - время [с], n - концентрация носителей заряда

Закон Ома для участка цепи

I = U / R
U = I·R
R = U / I

где U - напряжение [В], R - сопротивление [Ом]

Сопротивление проводника

R = ρ·l / S
ρ = R·S / l (удельное сопротивление)
R = R₀·(1 + α·Δt) (зависимость от температуры)

где ρ - удельное сопротивление [Ом·м], l - длина [м], S - площадь сечения [м²], α - температурный коэффициент

Последовательное соединение проводников

I = I₁ = I₂ = ... = I_n
U = U₁ + U₂ + ... + U_n
R = R₁ + R₂ + ... + R_n

Параллельное соединение проводников

U = U₁ = U₂ = ... = U_n
I = I₁ + I₂ + ... + I_n
1/R = 1/R₁ + 1/R₂ + ... + 1/R_n
R = (R₁·R₂) / (R₁ + R₂) (для двух резисторов)

Работа и мощность тока

A = q·U = I·U·t
P = A / t = I·U
P = I²·R = U² / R

где A - работа тока [Дж], P - мощность [Вт], t - время [с]

Закон Джоуля-Ленца

Q = I²·R·t
Q = I·U·t = U²·t / R

Количество теплоты, выделяемое проводником с током

Электродвижущая сила (ЭДС)

ε = A_стор / q
ε = I·R + I·r = U + I·r

где ε - ЭДС [В], r - внутреннее сопротивление источника

Закон Ома для полной цепи

I = ε / (R + r)
ε = I·R + I·r
U = ε - I·r (напряжение на внешней цепи)

Короткое замыкание

I_кз = ε / r
U = 0 (при коротком замыкании)

Соединение источников тока

ε_общ = ε₁ + ε₂ + ... + ε_n (последовательно)
r_общ = r₁ + r₂ + ... + r_n (последовательно)
ε_общ = ε (параллельно одинаковых источников)
1/r_общ = 1/r₁ + 1/r₂ + ... + 1/r_n (параллельно)

Правила Кирхгофа

ΣI_k = 0 (1-е правило: для узла)
Σε_k = ΣI_k·R_k (2-е правило: для контура)

где ΣI_k - алгебраическая сумма токов в узле, Σε_k - алгебраическая сумма ЭДС в контуре

Электрическая ёмкость

C = q / U
C = ε₀·ε·S / d (плоский конденсатор)
W = q²/(2C) = C·U²/2 = q·U/2 (энергия конденсатора)
ε₀ = 8.85·10⁻¹² Ф/м (электрическая постоянная)

Соединение конденсаторов

C = C₁ + C₂ + ... + C_n (параллельно)
1/C = 1/C₁ + 1/C₂ + ... + 1/C_n (последовательно)
C = (C₁·C₂) / (C₁ + C₂) (для двух последовательно)

Постоянный и переменный ток

I = I₀·sin(ωt) (переменный ток)
U = U₀·sin(ωt) (переменное напряжение)
I_действ = I₀/√2 (действующее значение)
U_действ = U₀/√2 (действующее значение)

Трансформатор

U₁/U₂ = N₁/N₂ = k (коэффициент трансформации)
I₁/I₂ = N₂/N₁ = 1/k (для идеального трансформатора)

где N₁, N₂ - числа витков обмоток, k - коэффициент трансформации

КПД источника тока

η = P_полезн / P_полн = R / (R + r)
η = U / ε

где η - коэффициент полезного действия
