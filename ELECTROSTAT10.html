ЭЛЕКТРОДИНАМИКА: ФОРМУЛЫ 10 КЛАСС

Закон Кулона

F = k · |q₁·q₂| / r²

где k = 9·10⁹ Н·м²/Кл², q₁ и q₂ - заряды, r - расстояние

Напряжённость электрического поля

E = F / q
E = k · |Q| / r² (для точечного заряда)

Работа электрического поля

A = q · (φ₁ - φ₂) = q · U

Напряжение и разность потенциалов

U = A / q = φ₁ - φ₂

Ёмкость конденсатора

C = q / U
C = ε₀ · ε · S / d (плоский конденсатор)

где ε₀ = 8.85·10⁻¹² Ф/м, ε - диэлектрическая проницаемость

Энергия заряженного конденсатора

W = C · U² / 2 = q² / (2C) = q · U / 2

Сила тока

I = q / t
I = U / R (Закон Ома)

Сопротивление проводника

R = ρ · l / S

где ρ - удельное сопротивление, l - длина, S - площадь сечения

Работа и мощность тока

A = I · U · t
P = I · U = I² · R = U² / R

Закон Ома для полной цепи

I = ℰ / (R + r)

где ℰ - ЭДС, r - внутреннее сопротивление
