Скорость света и показатель преломления

c = 3·10⁸ м/с (скорость света в вакууме)
n = c / v (абсолютный показатель преломления)
n₂₁ = n₂ / n₁ = v₁ / v₂ (относительный показатель преломления)
v = λ·ν (cвязь скорости, длины волны и частоты)

Закон преломления (Закон Снеллиуса)

n₁·sin α = n₂·sin β
n₂₁ = sin α / sin β

где α - угол падения, β - угол преломления

Полное внутреннее отражение

sin α₀ = n₂ / n₁ (при n₁ > n₂)

где α₀ - предельный угол полного внутреннего отражения

Тонкие линзы

1/F = 1/d + 1/f (формула тонкой линзы)
D = 1/F (оптическая сила линзы)
Γ = H/h = f/d = (F)/(d-F) (увеличение линзы)

где F - фокусное расстояние, d - расстояние от предмета до линзы,
f - расстояние от изображения до линзы, D - оптическая сила (в диоптриях)

Формула линзы через оптическую силу

1/d + 1/f = D
D = (n-1)(1/R₁ + 1/R₂) (формула для двояковыпуклой линзы)

Интерференция света

Δ = kλ (условие максимума)
Δ = (2k+1)λ/2 (условие минимума)
Δ = n·l (оптическая разность хода)

где Δ - разность хода, k = 0, ±1, ±2,... - порядок интерференции

Дифракция света

d·sin φ = kλ (условие главных максимумов для решётки)
d·sin φ = (2k+1)λ/2 (условие главных минимумов для щели)
λ = d·sin φ / k (для определения длины волны)

где d - период решётки, φ - угол дифракции, k - порядок спектра

Дифракционная решётка

N = 1/d (число штрихов на 1 мм)
φ_max ≈ tan φ = x/L (для малых углов)

где x - расстояние от центрального максимума до максимума k-го порядка,
L - расстояние от решётки до экрана

Дисперсия света

n = f(λ) (зависимость показателя преломления от длины волны)
n(фиол) > n(красн) (для нормальной дисперсии)

Поляризация света

I = I₀·cos² φ (закон Малюса)
tg αБ = n₂/n₁ (угол Брюстера)

где I₀ - интенсивность падающего поляризованного света,
I - интенсивность прошедшего света, φ - угол между плоскостями поляризации

Фотоны и энергия света

E = h·ν = h·c/λ (энергия фотона)
p = h/λ (импульс фотона)
h = 6.63·10⁻³⁴ Дж·с (постоянная Планка)
ħ = h/(2π) (редуцированная постоянная Планка)

Фотоэффект

hν = Aвых + (m·v²)/2 (уравнение Эйнштейна)
hν = Aвых + e·Uз
ν_min = Aвых / h (красная граница фотоэффекта)

где Aвых - работа выхода, Uз - задерживающее напряжение,
e = 1.6·10⁻¹⁹ Кл - заряд электрона

Давление света

p = (1+ρ)·I/c (при нормальном падении)
p = (2I)/c (при полном отражении)
p = I/c (при полном поглощении)

где I - интенсивность света, ρ - коэффициент отражения
