КОЛЕБАНИЯ И ВОЛНЫ: ФОРМУЛЫ 11 КЛАСС

Основные характеристики колебаний

x = A·cos(ωt + φ₀) (уравнение гармонических колебаний)
A - амплитуда колебаний [м]
T = t/N (период колебаний [с])
ν = N/t = 1/T (частота колебаний [Гц])
ω = 2πν = 2π/T (циклическая частота [рад/с])
φ = ωt + φ₀ (фаза колебаний)
φ₀ - начальная фаза

Скорость и ускорение при колебаниях

v = x' = -A·ω·sin(ωt + φ₀) (скорость)
v_max = A·ω (максимальная скорость)
a = v' = -A·ω²·cos(ωt + φ₀) (ускорение)
a = -ω²·x (ускорение пропорционально смещению)
a_max = A·ω² (максимальное ускорение)

Пружинный маятник

T = 2π√(m/k)
ω = √(k/m)
ν = 1/(2π)·√(k/m)

где m - масса груза [кг], k - жёсткость пружины [Н/м]

Математический маятник

T = 2π√(l/g)
ω = √(g/l)
ν = 1/(2π)·√(g/l)
l = g·(T/2π)² (длина маятника)

где l - длина маятника [м], g = 9.8 м/с² - ускорение свободного падения

Энергия колебаний

E = E_k + E_p (полная механическая энергия)
E_k = (m·v²)/2 = (m·A²·ω²·sin²(ωt+φ₀))/2 (кинетическая энергия)
E_p = (k·x²)/2 = (k·A²·cos²(ωt+φ₀))/2 (потенциальная энергия)
E = (k·A²)/2 = (m·A²·ω²)/2 (полная энергия)

Затухающие колебания

x = A₀·e^(-βt)·cos(ωt + φ₀)
ω = √(ω₀² - β²) (частота затухающих колебаний)
τ = 1/β (время релаксации)
λ = ln(A(t)/A(t+T)) = βT (логарифмический декремент затухания)

где β - коэффициент затухания, ω₀ - собственная частота

Вынужденные колебания. Резонанс

x = A_вын·cos(Ωt + φ_вын)
A_вын = F₀/(m·√((ω₀²-Ω²)² + 4β²Ω²))
Ω_рез = √(ω₀² - 2β²) (резонансная частота)

где Ω - частота вынуждающей силы, F₀ - амплитуда вынуждающей силы

Электромагнитные колебания

q = q₀·cos(ωt + φ₀) (заряд конденсатора)
i = q' = -q₀·ω·sin(ωt + φ₀) (сила тока)
U = q/C = (q₀/C)·cos(ωt + φ₀) (напряжение)

Колебательный контур

T = 2π√(L·C) (период колебаний)
ω = 1/√(L·C) (циклическая частота)
ν = 1/(2π√(L·C)) (частота)

формула Томсона, где L - индуктивность [Гн], C - ёмкость [Ф]

Энергия колебательного контура

W = W_э + W_м (полная энергия)
W_э = q²/(2C) = (q₀²/(2C))·cos²(ωt+φ₀) (энергия электрического поля)
W_м = (L·i²)/2 = (L·q₀²·ω²/2)·sin²(ωt+φ₀) (энергия магнитного поля)
W = q₀²/(2C) = (L·i₀²)/2 (полная энергия)

Механические волны. Основные характеристики

λ = v·T = v/ν (длина волны)
v = λ·ν (скорость волны)
ω = 2πν (циклическая частота)
k = 2π/λ (волновое число)

где λ - длина волны [м], v - скорость волны [м/с]

Уравнение бегущей волны

ξ(x,t) = A·cos(ωt - kx + φ₀)
ξ(x,t) = A·cos(2π(νt - x/λ) + φ₀)
ξ(x,t) = A·cos(2π(t/T - x/λ) + φ₀)

Стоячие волны

ξ(x,t) = 2A·cos(kx)·cos(ωt) (уравнение стоячей волны)
x_пучн = n·λ/2 (координаты пучностей, n = 0,1,2...)
x_узл = (2n+1)·λ/4 (координаты узлов, n = 0,1,2...)
l = n·λ/2 (длина струны для стоячей волны)

Звуковые волны

v_звука = √(γ·p/ρ) (в газах)
v_звука = √(E/ρ) (в твёрдых телах)
v_звука = √(K/ρ) (в жидкостях)
I = P/S = (ρ·v·ω²·A²)/2 (интенсивность звука)
L = 10·lg(I/I₀) (уровень громкости в дБ)
I₀ = 10⁻¹² Вт/м² (порог слышимости)

Эффект Доплера

ν' = ν·(v ± v_пр)/(v ∓ v_ист) (для механических волн)
ν' = ν·√((1 ± v/c)/(1 ∓ v/c)) (для световых волн)

где ν' - наблюдаемая частота, ν - частота источника, v_пр - скорость приёмника, v_ист - скорость источника

Скорость волны в различных средах

v = √(T/μ) (для струны)
v = √(E/ρ) (для стержня)
v_света = c/n (в среде)

где T - сила натяжения [Н], μ = m/l - линейная плотность [кг/м], E - модуль Юнга [Па]
