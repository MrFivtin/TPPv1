МАГНИТНОЕ ПОЛЕ: ФОРМУЛЫ 11 КЛАСС

Основные характеристики магнитного поля

B = F_max / (I·l) (индукция магнитного поля)
Ф = B·S·cos α (магнитный поток)
μ₀ = 4π·10⁻⁷ Гн/м (магнитная постоянная)

где B - магнитная индукция [Тл], Ф - магнитный поток [Вб], S - площадь [м²]

Сила Ампера (сила, действующая на проводник с током)

F_A = I·B·l·sin α
F_A = I·l·B (при α = 90°)

где I - сила тока [А], l - длина проводника [м], α - угол между проводником и вектором B

Сила Лоренца (сила, действующая на заряженную частицу)

F_Л = q·v·B·sin α
F_Л = q·v·B (при α = 90°)
F_Л = q·v·B = m·v²/R (для движения по окружности)

где q - заряд частицы [Кл], v - скорость частицы [м/с]

Движение заряженной частицы в магнитном поле

R = (m·v) / (|q|·B) (радиус траектории)
T = (2π·m) / (|q|·B) (период обращения)
ω = (|q|·B) / m (циклическая частота)

где R - радиус окружности [м], T - период [с], ω - угловая скорость [рад/с]

Магнитное поле прямого тока (закон Био-Савара-Лапласа)

B = (μ₀·μ·I) / (2π·r) (для бесконечно длинного прямого проводника)

где r - расстояние от проводника [м], μ - магнитная проницаемость среды

Магнитное поле в центре кругового витка

B = (μ₀·μ·I·N) / (2R)
B = (μ₀·μ·I) / (2R) (для одного витка)

где R - радиус витка [м], N - число витков

Магнитное поле соленоида (длинной катушки)

B = μ₀·μ·I·n
n = N / l (число витков на единицу длины)

где n - плотность витков [витков/м], l - длина соленоида [м]

Закон полного тока (теорема о циркуляции)

∮B·dl = μ₀·μ·ΣI

Циркуляция вектора B по замкнутому контуру равна алгебраической сумме токов, охватываемых контуром

Работа магнитного поля

A = 0 (при перемещении проводника с током или заряда по замкнутой траектории)
A = I·ΔФ (при изменении магнитного потока)

Магнитное поле не совершает работы над движущимся зарядом

Магнитный поток и закон электромагнитной индукции

ε_i = -ΔФ/Δt (закон Фарадея)
ε_i = -N·ΔФ/Δt (для катушки)
ε_i = B·l·v·sin α (для движущегося проводника)

где ε_i - ЭДС индукции [В], ΔФ - изменение магнитного потока [Вб], Δt - время изменения [с]

Индуктивность

L = Ф / I (индуктивность контура)
ε_s = -L·ΔI/Δt (ЭДС самоиндукции)
W_m = (L·I²)/2 (энергия магнитного поля катушки)

где L - индуктивность [Гн], ε_s - ЭДС самоиндукции [В], W_m - энергия [Дж]

Сила взаимодействия параллельных токов

F = (μ₀·μ·I₁·I₂·l) / (2π·d)

где d - расстояние между проводниками [м], l - длина проводников [м]

Магнитная проницаемость

μ = B / B₀

где B - индукция в среде, B₀ - индукция в вакууме

Магнитные свойства веществ

μ ≈ 1 (для парамагнетиков и диамагнетиков)
μ >> 1 (для ферромагнетиков)

Диамагнетики: μ < 1, Парамагнетики: μ > 1, Ферромагнетики: μ ≫ 1
