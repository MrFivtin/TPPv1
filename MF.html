МОЛЕКУЛЯРНАЯ ФИЗИКА: ФОРМУЛЫ 10 КЛАСС

Основные величины

ν = N / Nₐ = m / M (количество вещества)
Nₐ = 6.022·10²³ моль⁻¹ (постоянная Авогадро)
m₀ = M / Nₐ (масса одной молекулы)

Основное уравнение МКТ

p = (1/3) · n · m₀ · v²
p = (2/3) · n · E

где n = N/V - концентрация молекул, E - средняя кинетическая энергия

Температура и энергия

E = (3/2) · k · T (средняя кинетическая энергия поступательного движения)
v² = (3k·T) / m₀ (средний квадрат скорости)
k = 1.38·10⁻²³ Дж/К (постоянная Больцмана)

Уравнение состояния идеального газа (уравнение Менделеева-Клапейрона)

p · V = ν · R · T
p · V = (m / M) · R · T
R = 8.31 Дж/(моль·К) (универсальная газовая постоянная)

Частные случаи (газовые законы)

p · V = const (T = const) - закон Бойля-Мариотта
V / T = const (p = const) - закон Гей-Люссака
p / T = const (V = const) - закон Шарля
p · V / T = const - объединённый газовый закон

Изопроцессы

Изотермический: T = const, p · V = const
Изобарный: p = const, V / T = const
Изохорный: V = const, p / T = const

Внутренняя энергия идеального газа

U = (i/2) · ν · R · T

где i - число степеней свободы молекулы
i = 3 для одноатомного газа
i = 5 для двухатомного газа
i = 6 для многоатомного газа

Первое начало термодинамики

Q = ΔU + A

где Q - количество теплоты, ΔU - изменение внутренней энергии, A - работа газа

Работа газа

A = p · ΔV (при изобарном процессе)
A = ν · R · T · ln(V₂/V₁) (при изотермическом процессе)

Теплоёмкость

C = Q / ΔT
Cᵥ = (i/2) · R (молярная теплоёмкость при постоянном объёме)
Cₚ = Cᵥ + R = ((i+2)/2) · R (молярная теплоёмкость при постоянном давлении)
γ = Cₚ / Cᵥ (показатель адиабаты)

Уравнение теплового баланса

Qотд = Qполуч

Сумма отданных количеств теплоты равна сумме полученных

Влажность воздуха

φ = (ρ / ρ₀) · 100%
φ = (p / p₀) · 100%

где ρ - плотность водяного пара, ρ₀ - плотность насыщенного пара,
p - давление пара, p₀ - давление насыщенного пара
